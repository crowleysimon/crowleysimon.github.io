Guvera

Music Streaming Service

Details:

Guvera was a music streaming app with over 10 million downloads (on android alone).
I worked on it as a tester for about a year and then as an android developer for about two years. The app was written in Java and had an MVC architecture which was common at the time the project was started.
I had a hand in creating some of the features that would then go on to be used by millions of users, including some of the social features.

Technologies

  • Java
  • MVC
  • Butterknife
  • Retrofit
  • Jackson
  • Glide
  • Facebook
